**Amazon Web Scraping Project:**


![Web scraping](https://github.com/user-attachments/assets/83673dcd-5874-43df-b2dd-581665452f33)



This project scrapes product details from Amazon using Python and BeautifulSoup. 
The script extracts product titles, prices, ratings, review counts, and availability,
then saves the data into a CSV file for analysis.

**Features**

✅ Extracts product details from Amazon

✅ Saves data to a CSV file

✅ Handles missing values gracefully

**Technologies Used**


(Python,BeautifulSoup,Requests,Pandas,NumPy)

**How to Run??**


git clone https://github.com/rejinadel/Amazon-Web-Scraping-Project.git


cd Amazon-Web-Scraping-Project

**Install dependencies:**


pip install -r requirements.txt

**Run the script:**


python amazon_scraper.py
